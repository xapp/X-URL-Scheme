# X URL Scheme

X.app supports `x://`, `xapp://`, `x.app://`, `app.x.App://` and `xxx://` URL Schemes.

## `X` URL Schemes:
`x://`

`x://?videoURL=`


## `XAPP` URL Scheme:
`xapp://`

`xapp://?videoURL=`


## `X.APP` URL Schemes:
`x.app://`

`x.app://?videoURL=`



## `app.x.App` URL Scheme:
`app.x.App://`
This URL scheme is identical to bundleIdentifier of X.app

## `XXX` URL Scheme:
`xxx://`

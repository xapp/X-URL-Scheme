# X URL Scheme

X.app supports `x://`, `xapp://`, `x.app://`, `app.x.App://` and `xxx://` URL Schemes.

## `X` URL Schemes:
`x://`

`x://?videoURL=`


## `XAPP` URL Scheme:
`xapp://`

`xapp://?videoURL=`

Path and Option parameters are identical to `x://` URL Scheme


## `X.APP` URL Schemes:
`x.app://`

Path and Option parameters are identical to `x://` URL Scheme


## `app.x.App` URL Scheme:
`app.x.App://`

URL scheme is equal to BundleID of X.app.


## `XXX` URL Scheme:
`xxx://`

Backup URL scheme.
